Basic Git Commands

git init

git init
Initializes a new Git repository in the current directory.

git clone

git clone <repository-url>
Creates a copy of an existing repository from a remote source.

git add

git add <file>
Adds changes in the specified file to the staging area. 
git add . (all files at time)
Adds changes of all files to the staging area.

git commit

git commit -m "Your commit message"
Commits the staged changes to the repository with a descriptive message.

git status

git status
Displays the state of the working directory and staging area.

git push

git push <remote> <branch>
Uploads local repository content to a remote repository.

git pull

git pull <remote> <branch>
Fetches and integrates changes from a remote repository to the current branch.

git branch

git branch
Lists all local branches in the repository.

git checkout

git checkout <branch>
Switches to the specified branch.
git checkout -b <branch>
Switches to the newly created branch.

git merge

git merge <branch>
Merges the specified branch into the current branch.

git log

git log
Shows the commit history for the repository. 
git log --oneline
Shows the commit history for the repository in 1 line.

git remote

git remote -v
Lists the remote repositories associated with the local repository.
